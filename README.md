# 📈 台灣股票 PWA 看盤軟體 (TW-Stock-PWA)
> 一款專為手機單手操作設計、字體特大、零成本運行的行動台股看盤小工具。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ 特色亮點
- 🚀 **極致速度：** 每 20 秒自動同步臺灣證券交易所盤中數據。
- 📱 **類 App 體驗：** 支援 PWA 技術，一鍵「加入主畫面」全螢幕流暢運行，斷網亦可秒開介面。
- 📊 **動態視覺：** 內建大盤與個股的今日分時走勢圖（純 SVG 渲染，超輕量）。
- 🔋 **智慧省電：** 支援視窗休眠機制，切換視窗或關閉螢幕自動暫停輪詢，非盤中時間自動休眠。
- 💰 **無感成本：** 100% 前端運行，免伺服器、免資料庫、免登入、無廣告。

## 📸 線上體驗與安裝
👉 **即刻體驗網址：** https://smtmj2022.github.io/twstockapp

> **安裝教學 (Android / iOS)：**
> - **Android (Chrome):** 打開網址後，點擊瀏覽器右上方[...]選單「加到主畫面」彈窗。
> - **iPhone (Safari):** 點擊下方「分享」按鈕 ＞ 往下滑選擇「加入主畫面」。

## 🛠️ 開發與本地運行
本專案僅由 `index.html`、`manifest.json`、`sw.js` 組成，無任何複雜環境依賴。
1. 下載專案原始碼。
2. 在資料夾內執行：`python -m http.server 8000`
3. 瀏覽器開啟 `http://localhost:8000` 即可測試。
